## Images à remplacer plus tard

Les fichiers SVG sont des visuels temporaires. Ils peuvent être remplacés par tes captures réelles en conservant les mêmes noms, ou en modifiant les chemins dans `index.html`.

Dimensions recommandées pour les images de projets :

- 1 280 x 800 pixels ;
- ratio 16:10 ;
- format WebP ou PNG.

## Publication sur GitHub Pages

1. Crée un dépôt public nommé de préférence `Riad-cloud.github.io`.
2. Dépose tout le contenu de ce dossier à la racine du dépôt.
3. Ouvre `Settings`, puis `Pages`.
4. Dans `Source`, sélectionne `Deploy from a branch`.
5. Choisis la branche `main` et le dossier `/ (root)`.
6. Enregistre.
